Dynamic Image Gallery (React)

A simple and responsive image gallery built using React JS. This project demonstrates core React concepts such as reusable components, props, and dynamic rendering using the map() method.

Features
Reusable ImageCard component
Dynamic image rendering from data array
Clean and responsive grid layout
Easy to add new images
Component-based structure

Built With
React JS
JavaScript (ES6)
HTML5
CSS3

Images are stored in an array of objects and rendered dynamically using React. Each image is displayed using a reusable component, making the project scalable and easy to maintain.
